# Dat3Bot

Python bot that automates the tedious parts of popular dating apps ie. Tinder or Bumble

## Dependencies

- Windows 10
- python3
- selenium

## Getting Started

- Before use, you'll have to add Dat3Bot\chromedriver_win32 to your PATH variable.

## Run

```shell
python main.py
```
